## 流量运营

### 流量渠道诊断及策划（生意参谋）

#### 流量渠道诊断方法，锁定优化方向（流量）

解决问题：

- 判断高效率引流方式
- 判断优势引流渠道
- 辨别不理想引流数据

店铺前期重点关注数据：访客数、商品收藏人数、加购人数、支付转化率

| 操作               | 关联数据       | 优化方向                               |
| ------------------ | -------------- | -------------------------------------- |
| 访客降序           | 访客环比下降   | 找到原因并优化                         |
|                    | 支付转化率高   | 优势渠道，继续保持，多多发力           |
| 支付转化率降序     | 支付转化率下降 | 引起重视，找到原因并优化               |
|                    | 支付转化率高   | 高效率渠道，提升访客                   |
| 收藏人数和加购人数 | 收藏加购数     | 潜在转化客户多，加强产品投放，拉新操作 |

*注：实际需要同时考虑多个指标之间的影响和来源明细的实际内容，需要计算”期望值“，如：访客数×支付转化率。而非关注单一指标的高低。*



#### 确定竞品引流方式，锁定竞争渠道（竞争）

竞品分析三方面

- 竞品产品本身非常**优质**受欢迎，市场和消费者反馈比较好导致的流量多
- 竞争对手主要免费流量渠道的核心数据更好，这里的核心数据是指**主图点击率**和**转化率数据**
- 竞品的优势流量渠道是哪些，是否**可复制**

1. 竞品自身优质

   关键指标：**访客数**、引流渠道数、**收藏加购人气**

   竞品本身优体现为：①访客数多且稳定  ②入库数量多  ③收藏人气和加购人气高

   策略：更换蓝海市场、寻找产品差异点

2. 竞品核心数据好、免费流量渠道多

   关键指标：**点击率**、**免费流量支付转化指数**

   在免费流量上（手淘搜索、手淘推荐、淘内免费其他）竞品数据更好，说明对手的主图、主图视频、页面转化能力、客户评价反馈这些方面有优势。

   策略：点击率数据需要对竞品主图进行**点击率测试**，转化率数据通过**支付转化指数**分析，需要加强自身产品主图、点击率优化和产品转化率优化。**提升核心数据（点击率、产品转化率）可增加免费流量渠道**。

3. 竞品优势复制

   关键指标：**交易指数**

   交易指数高代表流量渠道创造的营业额高。产品单价相近时，证明流量渠道的访客和转化率客观。不仅需要看引流渠道，还需要调整数据时间维度，分析竞品在上升时期的流量结构，看是否有借鉴意义。



#### 高转化渠道提升引流效率（流量）

转化渠道对比分析：通过**渠道转化数据**和**产品转化数据**进行店铺转化归属对比分析，利于查漏补缺提升整体转化率。

渠道转化数据分析

获取流量来源构成（店铺来源），核心指标数据为：访客数、支付转化率、支付转化率变化。

1. 访客数较多且转化率高的流量入口。例如购物车、手淘旺信的转化率较高，可对应策略设立鼓励用户咨询、评价等活动，如咨询、晒图、问大家（互动）有礼活动，以此来增加转化率高的入口访客数。
2. 访客数较多但转化率下滑的流量入口，分析下滑原因，并指定优化策略。
3. 用转化率较高的产品参加<font color=red>每日好店</font>活动
4. 分析直播间转化率低的原因，并指定调整策略

产品转化数据分析（品类）

获取所有商品数据，核心指标数据为：访客数、平均停留时长、跳失率、收藏人数、加购件数、支付转化率。

1. 访客数多、转化率高的产品，是核心商品，需要每日观察、保证流量和转化数据
2. 访客数多、转化率低的产品，结合其他指标分析，分析客户流失原因
3. 计算收藏加购比，(收藏人数+加购件数)/访客数，其代表产品受欢迎程度、与转化率息息相关，若转化也高可加大引流力度。

### 手淘搜索影响因素

#### 手淘搜索影响因素（品类）

核心因素：类目相关性、属性相关性、关键词相关性、价格相关性、产品销量、产品数据表现

1. 商品类目、属性、价格决定了商品的基础曝光人群。以此为基础加消费者点击浏览行为形成关键词搜索量

2. 保证产品标题关键词的有效性，其是获取搜索流量的端口来源。关键词**需要是市场有搜索量的关键词**

   可在【生意参谋】-【市场】-【搜索排行】中选择符合商品本身并且有一定搜索量的关键词进行使用。

3. 产品数据表现力，代表了消费者对产品的喜好度。浏览行为（点击率、停留时长、收藏加购率等）、转化行为（转化率、客单价、UV价值等）、售后行为、复购行为等。

#### 优化产品标题（品类）

1. 通过【品类】-【商品360】找到低效关键词
2. 寻找市场优质关键词【市场】-【搜索分析】，在相关分析中寻找符合商品本身的优质关键词
3. 修改标题，删除低效词根、替换新词根。将优质关键词拆分为词根，如“连衣裙修身气质 收腰”，拆分为“连衣裙”、“修身”、“气质”、”收腰“，选取词根加入标题中。

#### 搜索场景搭建，获取有效搜索流量

搜索场景：根据消费着搜索浏览行为，动态匹配推荐给消费者需求商品。

构建搜索场景三要素：①搜索关键词  ②消费人群  ③产品本身

1. 搜索关键词代表不同的消费需求

2. 选择匹配产品的消费人群。【直通车】-【流量解析】可以获取搜索关键词的人群画像特征

   搜索关键词+消费者人群本身进行匹配对应的产品 = 搜索场景搭建

   通过数据的正向反馈来判断匹配的有效性



### 手淘推荐影响因素

#### 掌握推荐资源位

推荐资源位包含：猜你喜欢、猜你喜欢微详情、红包互动权益场

- 猜你喜欢包含：首页推荐（购前）、购物车收藏夹（购中）、待付款待收货待发货（购后）

  对于购买决定周期较长的类目，如家具，存在着较多产品之间的比较，做购中的猜你喜欢效果更好。

  对于退货率较高、复购周期短的类目，如服装、零食，购后的运营可能会有奇效。

- 微详情★：根据算法匹配给客户不停推荐感兴趣的商品，而不关乎类目。根据算法标签匹配来决定。如果商品有接近的跨类目人群，或本类目的竞争激烈，微详情就有优势。如：教育类目，其与图书、3C人群高度接近。

- 红包互动权益场：流量来源是淘宝的打卡小活动。流量人群两极分化，为羊毛党或淘宝深度用户。适合低客单和高客单两极的小众商品进行投放。女性人群也是该资源位的共通点，适合女性类目。

#### 手淘推荐核心影响因素

目前，手淘推荐流量 >> 手淘搜索流量

千人千面推荐算法：根据性别、年龄、地域、收入等基础数据推荐商品，然后根据用户行为（浏览、点击、停留、收藏、加购、咨询、购买、评价、回购）进一步判断需求。

五个核心因素：

1. 商品标签精准

   推荐流量属于物找人，会根据产品的成交、收藏、加购、点击、标题、价格、属性、回购，来判断推荐目标人群。所以商品前期的各项数据要是精准的目标客户人群，并且标题、价格、属性、回购不要大幅度修改，因为这些数据直接影响到后续流量的精准性。

2. 商品标签入池

   入池：是让系统知道产品特征，与哪些产品是一样的，适合哪些人群。入池的判断依据主要是商品的成交、收藏加购、复购、点击等等。需要积累到一定的数据量才能入池。数据量达到多少未知。可以通过【鹿班工具】-【主图点击优化】-【首页猜你喜欢】查看店铺入池商品的个数。

   入池方法：一般是用付费流量精准加大点击、成交、收藏、加购、复购等。

3. 商品主图点击率高

   图片点击率是最核心考量指标。淘宝分为实时赛马和长期赛马。点击率对实时赛马有很大影响，点击率越高，相同展现下获得的流量就越多，对于宝贝权重的提高也就越大。

   高点击率主图要素：①浅色背景、无牛皮癣、少文字、无边框、图片清晰  ②多看多学习，同行手淘推荐流量较多的产品

4. 商品转化率高

   转化率反映商品的综合运营能力，转化率过低会造成UV价值偏低，流量浪费大。系统在推荐考核时，会考虑流量产值最大化，所以会考虑转化率。如果转化率过低，推荐流量也会受到压制。

   提高转化率方法：①付费精准的持续推广  ②标题词根、商品主图、卖价跟流量人群相匹配

5. 商品收藏加购人气高

   收藏加购率长期稳高于同行有助于长久获取推荐流量，所以要多做引导用户收藏加购的行为。

但不是所有类目都适合做手淘推荐，可以参考竞店的流量渠道来源。

① 持续生产并优化主图视频和主图图片可以帮助商品在微详情内获得更多推荐机会。

② 设置装修微详情页面，设置成功后会自动流转到系统算法推荐池

③ 好的竖屏图和视频（点击率高）会提升商品卡片推荐机会。

#### 掌握人群标签，提升推荐流量

人群标签分两大类：①基础标签：年龄、性别、消费能力等  ②行为标签：浏览过、购买过、收藏过

基础标签起微调作用，行为标签是核心作用。

标签在各场景下的应用：

1. 基于宝贝推荐：相似的宝贝推荐给需要的用户

3. 基于标签场景的推荐：推荐该类目的周边产品，如浏览过奶粉会推荐奶瓶

### 直通车

#### 直通车选词

1. 定范围

   从【生意参谋】-【市场洞察】获取该类目下“热搜”和“飙升”的搜索词，得到两个词组中的交集部分并过滤出符合自身商品的词。

   或：直通车的【词推荐】中。①直接从竞品优质词选取，指标筛选优先级依次为（点击转化率、相关性、展现指数>2000）；②综合推荐词建议宝贝已有过百左右的真实销量再使用（竞争强，不适合新品）

2. 质量分验证

   将选中的词添加到关键词中，等待1~2分钟刷新，删除移动端质量分低于7分的，若9~10分的词大于3个，则删除其他词。剩下的词是需要投放验证测试的词。

3. 投放验证

   设置词出价，电脑端0.05、移动端排名前10的位置，设置预算。等预算花完后，查看关键词的点击率情况。保留点击率大于市场平均的词，若没有则保留点击率排名前三的词。得到选词对象。

4. 定词

   确定选词之后，做人群定向溢价，进一步优化词的精准度。

#### 关键词出价技巧，确定出价上限

- 直通车扣费的本值

  影响排名位置的变量有：点击率、点击量、转化率等，所以操盘时应关注平均展现排名。

- 影响扣费的参数

  ①分时折扣：根据自定义时间段，设置出价

  ②关键词出价：其决定了流量规模的大小，会受分时折扣的影响，即自己设定的价格

  ③精选人群溢价：在关键词基础上，继续对某特定人群做细分

  ④智能出价：根据出价目标和流量动态，系统自动提升高质量流量溢价，缩减低质量流量的溢价。即开启智能出价功能

  ⑤抢位助手：自动调整关键词出价、抢占并保持排名的工具。设置抢位策略、及时调整关键词的出价。

- 直通车出价技巧

  | 影响出价的因素 | 出价范围   |
  | -------------- | ---------- |
  | 分时折扣       | 30%~250%   |
  | 关键词出价     | 0.05~99.99 |
  | 精选人群溢价   | 5%~300%    |
  | 智能出价       | 30%~300%   |
  | 抢位助手       | 1%~200%    |

  $$
  实际出价 = 分时折扣×出价×(1+智能出价)×(1+人群溢价)×(1+抢位助手)

#### 报表分析

报表类型

1. 账户报表：整体数据展现
2. 计划报表：每个推广计划的数据
3. 推广单元报表（宝贝报表）：计划由多个推广单元组成，每个宝贝的推广数据

6. 地域报表：推广单元包含多个地域，呈现了地域维度的数据

报表分析

时间维度：选择的转化周期越长，数据可行度越高。

新创建计划

​	新创建资源位溢价尽可能使用统一溢价，如30%。可以观察同一组人群在不同资源位上的具体表现。

成熟型计划

​	对于已经产生了资源位数据，根据各个资源位的点击率、加购成本、投资回报数据，对不同资源位进行个性化溢价。单次调整比例控制在15%以内。



#### 无展现排查与解决

1. 人群选择，决定了计划流量的精准度和广度，查看”投放信息提示“指示图，确保定向人群范围适中，无人群过于狭窄。
2. 出价调整，设置建议出价的70%~100%，获取第一轮展现量。后期再调价。
3. 调大溢价设置。人群溢价（智能调价）30%；资源位溢价30%
4. 调整计划时间折扣，每次调整折扣控制在15%以内。



## 名词概念

### PV（Page View)

页面访问量。即页面浏览量或点击量，衡量网站用户访问的网页数量；在一定统计周期内用户每打开或刷新一个页面就记录1次，多次打开或刷新同一页面则浏览量累计。

### UV(Unique Visitor)价值

UV价值=销售额/访客数

销售额=访客数\*转化率\*客单价

∴ UV价值=转化率*客单价

Unique Visitor独立访客，统计1天内访问某站点的用户数(以cookie为依据);访问网站的一台电脑客户端为一个访客。可以理解成访问某网站的电脑的数量。网站判断来访电脑的身份是通过来访电脑的cookies实现的。如果更换了IP后但不清除cookies，再访问相同网站，该网站的统计中UV数是不变的。如果用户不保存cookies访问、清除了cookies或者更换设备访问，计数会加1。00:00-24:00内相同的客户端多次访问只计为1个访客。

### 低客单、高客单

每单价格较低、每单价格较高

### PPC平均点击花费（Pick Promote Ctr）

PPC = 花费 ÷ 点击量

### ROI投资回报率(Return On Investment)
